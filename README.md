## 🚀 Run Project

1️⃣ Clone project and access its folder:

```bash
$ git clone https://github.com/jtiagosantos/to.do.git
$ cd to.do
```

2️⃣ Install dependencies:

```bash
$ yarn ou npm i
```

3️⃣ Start project:

```bash
$ yarn dev ou npm run dev
```
